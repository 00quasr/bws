# NetPulse User Manual Screenshots

This directory contains screenshots for the NetPulse User Manual.

## Capturing Screenshots

To capture the screenshots, run the interactive capture script:

```bash
cd /path/to/NetPulse/docs/manual
./capture_screenshots.sh
```

The script will guide you through capturing each screenshot interactively.

## Screenshot Guidelines

- **Resolution**: Capture at 1920x1080 or higher
- **Format**: PNG with transparency where applicable
- **Theme**: Use the default/light theme for consistency
- **Content**: Include sample data (hosts, alerts) for realistic appearance
- **Focus**: Ensure the relevant UI element is clearly visible
- **Annotations**: Do not add annotations to screenshots; they are added in the manual
